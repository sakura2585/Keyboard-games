# 鍵盤小遊戲

給小朋友練習**英數按鍵**：看畫面上的字，按下對應鍵，累積分數。

## 怎麼跑

### 桌面版（Python / tkinter）

```powershell
cd "H:\PY\鍵盤小遊戲"
python main.py
```

### 網頁版（推薦部署）

```powershell
cd "H:\PY\鍵盤小遊戲\web"
# 直接雙擊 index.html 也可，或用本機伺服器：
python -m http.server 8000
```

開啟瀏覽器到 `http://localhost:8000`。

**請用本機伺服器**（如上），不要只靠雙擊 `index.html`，否則瀏覽器可能無法載入 `data/vocabulary.json`。

## 辭彙庫（可編輯）

- 檔案位置：`web/data/vocabulary.json`（JSON 格式，可用記事本或 VS Code 編輯）。
- **`wordPool`**：給「簡單英文單字」；每筆 `{ "en": "僅小寫英文", "zh": "中文說明" }`，可選 `"topic"`、`"img"`（相對 `web/` 的圖片路徑）。**國家**類預設會顯示 `web/images/flags/{en}.svg` 於題目左欄（可改 `.png` 等請在該筆加 `img`）；無圖或載入失敗時左欄仍為「已完成」。
- **`bopomofoWordPool`**：給「注音拼字」；每筆 `{ "han": "顯示用中文", "bop": "注音串" }`，須符合本專案注音鍵位，音節之間可空格。
- 若檔案缺失、格式錯誤或某一區陣列為空，程式會改用程式內建預設辭彙。
- **桌面版** `main.py` 的「字詞」模式會從同一檔讀取 `wordPool` 的英文單字；改完存檔後請重新啟動桌面程式。

## 需要什麼

- 桌面版：Python 3.10+（建議），且安裝內含 **tkinter**。
- 網頁版：任一現代瀏覽器（Chrome / Edge / Firefox）。

## 操作

1. 選模式（英數單鍵 / 純英文 / 簡單英文單字 / 注音模式 / 注音拼字）；可開關**顯示模擬鍵盤**。
2. 集中看上方目標，按下鍵盤對應鍵。
3. 英文不分大小寫；答錯會提示，不扣分。
4. 鍵盤提示色：黃色（應按）、綠色（按對）、紅色（按錯）；**應按的鍵**另有加粗外框。模擬鍵盤底色為**標準 QWERTY 盲打手指分區**（空白鍵為拇指區）；**F、J** 為基準列定位鍵，同色調略加深。
5. **音效**：可勾選**按鍵音效**（答對／答錯短音，存於偏好）；本局結束與**該模式破紀錄**另有提示音（與按鍵音效開關無關）。若瀏覽器無聲，可先點一下頁面再試。
6. **簡單英文單字**：會顯示中文翻譯；**發音**先勾選主開關後，再以**下拉選單**選「成為題目發音」或「完成題目發音」；**語速**可另選。**發音（目前題目）**按鈕隨時手動播放，不受自動發音開關影響。
7. **英文詞彙種類**：僅在簡單英文單字模式顯示；標題列可點擊**展開／收合**，收合時仍會顯示目前為「全部」或「已選 N 種」。
8. **本局結束**：可按 **Enter** 或點**再玩一局**；結束畫面開啟時焦點會在按鈕上，方便鍵盤操作。畫面內可檢視**分數紀錄表**（最佳 10 筆與最後一局，含正確／錯誤／連擊等）。
9. **紀錄表**：隨時可點右上角 **紀錄表** 開啟（不佔主畫面）；**五種模式各有獨立紀錄**，用 **紀錄模式** 下拉切換。於紀錄檢視中按 **Esc** 或 **關閉** 返回。

## 常見問題

- **開不起來、說沒有 tkinter**：改用 Windows 官方安裝的 Python，或重裝時勾選 tcl/tk 元件。
- **桌面版視窗跑出螢幕**：刪掉專案內 `state/app_state.json` 後重開。
- **網頁版按鍵沒反應**：先點一下網頁空白處讓頁面取得焦點，再按鍵。
- **改了 vocabulary.json 網頁沒變**：請強制重新整理（Ctrl+F5）；若用 `file://` 開網頁，請改為在 `web` 目錄跑 `python -m http.server` 再開 `http://localhost:8000`。

## 設定檔

- 桌面版：`state/app_state.json`（視窗位置／大小、上次難度，不提交到 Git）。
- 網頁版：瀏覽器 `localStorage` 鍵名 **`typing-game-web`**（JSON），大致包含：模式、`speakWhen`（自動發音 off／時機）、語速、是否顯示模擬鍵盤、**按鍵音效**（`sfxKeyEnabled`）、起始能量遞減、英文 topic 篩選、詞彙種類面板是否展開等。實際欄位以 `web/app.js` 的 `savePrefs()` 為準。
- 另 **`typing-game-web-score-history`**：依模式分區儲存（`v:2` 與 `byMode`，見 `mergeScoreArchive`／`loadScoreArchive`）。

- 上傳 **`web/`** 目錄內容即可（含 `index.html`、`style.css`、`app.js`、`data/`、`images/`）。
- 勿依賴 `file://` 開啟（辭彙 JSON 需 HTTP）；GitHub Pages 等靜態託管可直接指向 `web` 為網站根目錄。
