# 鍵盤小遊戲

給小朋友練習**英數按鍵**：看畫面上的字，按下對應鍵，累積分數。

## 怎麼跑

### 桌面版（Python / tkinter）

```powershell
cd "H:\PY\鍵盤小遊戲"
python main.py
```

### 網頁版（推薦部署）

```powershell
cd "H:\PY\鍵盤小遊戲\web"
# 直接雙擊 index.html 也可，或用本機伺服器：
python -m http.server 8000
```

開啟瀏覽器到 `http://localhost:8000`。

**請用本機伺服器**（如上），不要只靠雙擊 `index.html`，否則瀏覽器可能無法載入 `data/vocabulary.json`。

## 辭彙庫（可編輯）

- 檔案位置：`web/data/vocabulary.json`（JSON 格式，可用記事本或 VS Code 編輯）。
- **`wordPool`**：給「簡單英文單字」；每筆 `{ "en": "僅小寫英文", "zh": "中文說明" }`，可選 `"topic": "稱謂|動物|…"` 方便自己整理（遊戲隨機出題時可不使用 topic）。
- **`bopomofoWordPool`**：給「注音拼字」；每筆 `{ "han": "顯示用中文", "bop": "注音串" }`，須符合本專案注音鍵位，音節之間可空格。
- 若檔案缺失、格式錯誤或某一區陣列為空，程式會改用程式內建預設辭彙。
- **桌面版** `main.py` 的「字詞」模式會從同一檔讀取 `wordPool` 的英文單字；改完存檔後請重新啟動桌面程式。

## 需要什麼

- 桌面版：Python 3.10+（建議），且安裝內含 **tkinter**。
- 網頁版：任一現代瀏覽器（Chrome / Edge / Firefox）。

## 操作

1. 選難度（英數單鍵 / 純英文 / 簡單英文單字 / 注音模式 / 注音拼字）。
2. 集中看上方目標，按下鍵盤對應鍵。
3. 英文不分大小寫；答錯會提示，不扣分。
4. 鍵盤提示色：黃色（應按）、綠色（按對）、紅色（按錯）。
5. 英文單字模式會顯示中文翻譯；完成單字可自動發音（可開關與調語速）。

## 常見問題

- **開不起來、說沒有 tkinter**：改用 Windows 官方安裝的 Python，或重裝時勾選 tcl/tk 元件。
- **桌面版視窗跑出螢幕**：刪掉專案內 `state/app_state.json` 後重開。
- **網頁版按鍵沒反應**：先點一下網頁空白處讓頁面取得焦點，再按鍵。
- **改了 vocabulary.json 網頁沒變**：請強制重新整理（Ctrl+F5）；若用 `file://` 開網頁，請改為在 `web` 目錄跑 `python -m http.server` 再開 `http://localhost:8000`。

## 設定檔

- 桌面版：`state/app_state.json`（視窗位置／大小、上次難度，不提交到 Git）。
- 網頁版：使用瀏覽器 `localStorage` 記住上次難度。
